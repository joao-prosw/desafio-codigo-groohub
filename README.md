# Desafio de projeto da Groohub #

### Necessário para rodar o código:  
#### 1- .NET SDK pelo menos na versão 10;
#### 2- SQL Server 2025 Express;
#### 3- Ao clonar o repo, acessando ele no Visual Studio Code, primeiro instale a tool global do entity framework com o seguinte comando de terminal no diretório do projeto: `dotnet add package Microsoft.EntityFrameworkCore.Tools` e `dotnet tool install --global dotnet-ef`;
#### 4- Para lançar as migrations em banco de dados rode `dotnet ef database update`;
#### 5- Após isso, para iniciar o projeto, execute no terminal `dotnet run` ou `dotnet watch`.
